
# 📊 Data Analytics Project Planning using Network Diagram & Gantt Chart

## 📌 Project Overview

This project demonstrates **project planning and scheduling techniques** for a data analytics initiative using:

- **Network Diagram**
- **Critical Path Analysis**
- **Slack Time Calculation**
- **Gantt Chart Planning**

The objective was to develop a structured project schedule, identify critical tasks, calculate project duration, and visualize dependencies using project management techniques.

---

## 🎯 Objective

To plan and manage a **data analytics project lifecycle** by:

- Defining task dependencies
- Identifying the **critical path**
- Calculating **slack time**
- Creating a **Gantt Chart timeline**
- Estimating project duration

---

## 🔗 Task Dependencies

Key dependencies include:

- Data cleaning dependent on data collection
- Predictive model dependent on EDA
- Final integration dependent on:
  - Data Integration
  - Model Validation
  - Feedback Integration
